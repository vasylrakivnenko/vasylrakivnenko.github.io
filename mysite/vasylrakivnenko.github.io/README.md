# Personal Website (Static)

This is a simple, single-page personal site (HTML/CSS/JS) that works great with **GitHub Pages**.

## Files
- `index.html` — content
- `styles.css` — styling (dark + light theme)
- `script.js` — menu toggle, theme toggle, and a tiny contact-form helper

## Quick deploy on GitHub Pages
1. Put these files in the root of your repo (or in `/docs`).
2. On GitHub: **Settings → Pages**
3. **Build and deployment**
   - Source: *Deploy from a branch*
   - Branch: `main`
   - Folder: `/ (root)` (or `/docs` if you chose that)
4. Save — your site will be live on the Pages URL.

## Customize
- Update text + links inside `index.html`
- Replace placeholder portfolio links and the email `mailto:...`
- Add images under `assets/` and update the Photography gallery
